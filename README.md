# Mobile Application Lab — Record

A simple repository containing lab experiments and exercises for the Mobile Application Development course. This repository includes Kotlin-based Android projects along with brief notes for each experiment.

## Purpose
To maintain a chronological and organized record of lab work, including:

Practical implementations

Mini demos

Experiment-wise learning outcomes

## Tech
- Kotlin

- Android Studio

- Android SDK

## How to run
1. Open the project in Android Studio

2. Allow Gradle to sync

3. Click Run 

Execute on an emulator or physical Android device
## Project Structure
- app/ — Android module (source, resources, tests)
- notes/ — optional lab notes or screenshots
